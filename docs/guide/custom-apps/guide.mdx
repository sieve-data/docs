Let's start by creating an echo function that adds an exclamation mark to the string that you pass in! 

### Create a python file

Create a python file for your echo function.

```bash
touch echo.py
```

### Create the sieve function

Copy the following content into the python file.

```python
import sieve

@sieve.function(
    name="echo_with_exclamation",
)
def echo_with_exclamation(s: str) -> str:
    return s + "!"
```

### Run the function

Create another python file to run the function 

```bash
touch main.py
```

Copy paste the following code that runs the sieve function


```python
import sieve

echo_with_exclamation = sieve.function.get(f"<your-org-here>/echo_with_exclamation")
result = echo_with_exclamation.run("hello world")
print(result)
```

Run the file and see the results!

```bash
$ python main.py
hello world!
```
