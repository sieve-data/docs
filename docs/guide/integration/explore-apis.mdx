Sieve provides access to highly optimized AI video APIs for [hundreds of use cases](https://sievedata.com/explore).
These APIs are designed to be easy to integrate as out-of-the-box solutions while giving you flexibility and control over parameters.

For detailed information, please refer to each API's [usage guide](https://sievedata.com/functions/sieve/autocrop/guide).
